# AIS-2
# Чат-бот для курса Системы Искуственного Интеллекта

## Область

Данный чат-бот разработан для помощи в поиске статей и журналов. 
Чат бот умеет советовать статьи, искать подходящие по параметрам статьи, искать журналы, показывать требования, 
предлагать журналы для публикации.

Поговорить с ботом можно по [ссылке](http://example.com/).

## Сценарии бесед 
На данный момент бот поддерживает 3 варианта диалогов.
* справочные разговоры (команды)
* диалог про статьи
* диалог про журналы

## Примеры бесед

| Собеседник        | Успех  | Неудача |    Ключевые слова    |  
| ------------- |:-------------:| :-------------:| :-------------:|
| {Приветствие}        | {Предложение посмотреть команды или начать диалог} | - |   * |
| Найти статью       | Это конкретная статья?  | - | найти + статья | 
| Статья \[название]      | \[статья]  | {Сообщения об ошибке} |   статья |
| Статья \[doi]      | \[статья]  | {Сообщения об ошибке} |   статья |
| Статья \[автор]      | \[статья]  | {Сообщения об ошибке}  |   статья |
| Статья из \[журнал]      | \[статья]  | {Сообщения об ошибке}  |   статья + из|
| Авторы \[статья]     | \[Авторы]  |{Сообщения об ошибке} | автор |
| Цитирования \[статья]     | \[Число]  | {Сообщения об ошибке} | цитирование|| Самая \[Тип] статья     | \[статья]  | {Сообщения об ошибке} | лучший/популярный/новый + статья |
| Статья про \[тематика]   или  Статья о \[тематика]  | \[статья]  | {Сообщения об ошибке}  |   статья + про |

### Шаблоны

* статьи
    * статья про \[тематика], статья о \[тематика]
    * статья \[автор]
    * статья из \[журнал]
    * статья \[название]
    * статья \[DOI] 
    * авторы \[статья]
    * цитирования \[статья]


## Список синонимов 

Для составления списка синонимов использовался ресурс [КартаСлов](https://kartaslov.ru/).

| Слово        | Синонимы           
| ------------- |:-------------:|
| найти  | находить, отыскивать, откапывать,раздобыть, узнать, разыскать, подобрать, выудить, вытащить, выяснить, определить, установить |
| статья   | публикация, книга, работа, труд, произведение, отчёт, доклад, выработка,  диссертация      |
| требования | правила, рекомендации, регламент |
| автор | составитель |
| цитирование | ссылка, цитата |
| лучший | хороший, превосходный, профессиональный, замечательный, выдающийся, достойный |
| новый | последний, недавний,  крайний, свежий |
| да | верно, правильно, отлично, точно, определённо, ясно, ага, угу|
| шутка | анекдот, шуточка, прикол |
